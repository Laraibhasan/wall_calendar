# 📅 Interactive Wall Calendar Component

A polished, responsive **wall calendar web component** built using React, inspired by a physical hanging calendar design.

🔗 **Live Demo:** https://wall-calendar-eight-zeta.vercel.app/

---

## ✨ Features

* 📆 **Dynamic Calendar**

  * Accurate month layouts (aligned with real-world dates)
  * Monday-first calendar view
  * Supports all months dynamically

* 🔁 **Month Navigation**

  * Smooth animated transitions between months
  * Intuitive arrow controls

* 🎯 **Date Range Selection**

  * Select start and end dates
  * Highlighted range with clear visual states

* 📝 **Integrated Notes Section**

  * Add monthly notes
  * Clean and distraction-free UI

* 🎨 **Wall Calendar Aesthetic**

  * Hanging pin + spiral effect
  * Hero image per month
  * Seasonal images for all 12 months
  * Diagonal design accents

* 📱 **Fully Responsive**

  * Mobile-first layout
  * Stacked design on smaller screens
  * Optimized tap targets

* 📍 **Current Date Indicator**

  * Highlights today’s date for better usability

---

## 🛠 Tech Stack

* **React (Vite)**
* **Tailwind CSS**
* **Framer Motion** (animations)
* **date-fns** (date handling)
* **Lucide React** (icons)

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/Laraibhasan/wall_calendar.git
cd wall_calendar
```

---

### 2. Install dependencies

```bash
npm install
```

---

### 3. Run the development server

```bash
npm run dev
```

Then open:

```text
http://localhost:5173
```

---

## 📦 Build for production

```bash
npm run build
```

---

## 💡 Design & Implementation Notes

* The layout is inspired by a **physical wall calendar**, combining imagery and functionality.
* Calendar grid maintains a **fixed 6-row structure** to prevent layout shifts.
* Month images dynamically change to match **seasonal themes**.
* Animations are intentionally subtle to maintain a **premium UX feel**.
* No backend is used — all logic is handled client-side.
